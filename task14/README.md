Type Detector in Go
Простая программа для определения типов переменных во время выполнения (runtime) на языке Go.

Описание
Программа демонстрирует два подхода к определению типов переменных:

Использование type switch для базовых типов

Использование рефлексии (reflect) для определения каналов

Возможности
Определение базовых типов: int, string, bool, float64

Определение срезов: []int

Определение каналов любых типов с помощью рефлексии

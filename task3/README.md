# Worker Pool на Go (канал + N воркеров)

Небольшая учебная программа на Go:

- **Главная горутина (producer)** постоянно отправляет данные в канал.
- **N воркеров (горутины)** читают данные из этого канала и выводят их в `stdout`.
- Количество воркеров задаётся параметром `-workers`.
- Есть параметр `-buffer` для размера буфера канала.
- При нажатии **Ctrl+C** программа корректно завершает работу:
  - producer закрывает канал,
  - воркеры дочитывают остаток и завершаются,
  - `main` ждёт всех воркеров через `WaitGroup`.

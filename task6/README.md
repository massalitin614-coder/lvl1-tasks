Go Горутины: Как остановить
Простая демонстрация разных способов остановки горутин.

Запуск
bash
go run main.go
Методы остановки:

1. Сама заканчивает работу - работает пока не выполнит условие

2. По команде через канал - внешний код говорит "стоп"

3. Через контекст - для HTTP запросов

4. Goexit() - немедленная остановка

5. Закрытие канала - для воркеров

6. По времени - таймаут

7. По Ctrl+C - системный сигнал

8. Паника с восстановлением - аварийная остановка

9. Контекст с таймаутом - комбинированный способ

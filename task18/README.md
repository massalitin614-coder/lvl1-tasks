Потокобезопасный счётчик на Go
Простая реализация структуры-счётчика, который безопасно работает в конкурентной среде (из нескольких горутин).

Особенности
Потокобезопасность: Использует sync.RWMutex для безопасного доступа из множества горутин

Оптимизация чтения: RWMutex позволяет множественное одновременное чтение

Простой API: Четыре базовых метода для работы со счётчиком

Корректность: Гарантирует точный подсчет даже при высокой нагрузке

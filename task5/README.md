Producer-Consumer с таймаутом на Go
Описание
Программа демонстрирует паттерн Producer-Consumer с использованием каналов и горутин. Writer отправляет числа в канал, Reader их читает. Через N секунд программа автоматически завершается.

Функциональность
Producer: Генерирует числа 1,2,3... каждые 500мс

Consumer: Читает числа из канала

Таймаут: Автоматическое завершение через заданное время

Синхронизация: Корректное завершение всех горутин

Технологии
Горутины для конкурентности

Каналы для коммуникации

Context для управления временем

WaitGroup для синхронизации

Запуск
bash
go run main.go

Настройка
Измените N в main() для другого времени работы

Измените time.Sleep() для другой скорости отправки

Измените размер буфера канала при создании

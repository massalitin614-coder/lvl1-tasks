Описание
Программа вычисляет квадраты чисел из массива [2, 4, 6, 8, 10] с использованием конкурентности в Go. Каждое вычисление выполняется в отдельной горутине.

Особенности реализации
Используемые технологии:
Горутины (Goroutines) для параллельных вычислений

Каналы (Channels) для безопасной передачи данных между горутинами

sync.WaitGroup для синхронизации завершения горутин

Буферизированный канал предотвращает блокировки

Преимущества данного решения
Параллелизм: вычисления выполняются одновременно

Безопасность: каналы обеспечивают безопасную передачу данных

Эффективность: буферизация предотвращает блокировки

Чистота кода: использование defer и range делает код читаемым

Масштабируемость: легко адаптировать для большего количества чисел

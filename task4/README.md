# Worker Pool на Go с Graceful Shutdown

Простая программа, демонстрирующая пул воркеров на Go с корректным завершением по сигналам операционной системы.

## Как запустить

1. Убедитесь, что у вас установлен Go (версии 1.16 или выше).
2. Скопируйте код в файл `main.go`.
3. Запустите в терминале:
   ```bash
   go run main.go
